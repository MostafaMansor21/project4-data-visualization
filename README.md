# Project 4 - Data Visualization

## Overview
This project focuses on transforming e-commerce data into clear visual insights using Excel and PowerPoint.

## Objectives
- Create effective data visualizations
- Identify business insights
- Apply data storytelling principles
- Support decision-making through visual analysis

## Visualizations
- Monthly Revenue Trend
- Order Status Distribution
- Payment Method Distribution

## Key Findings
- Sales consistently peak during May and June.
- Payment methods are evenly distributed across customers.
- Cancelled and returned orders present opportunities for operational improvement.

## Recommendations
- Increase marketing efforts before peak sales periods.
- Improve fulfillment processes to reduce cancellations and returns.
- Maintain support for multiple payment methods.
- Improve data quality and promotional tracking.

## Tools Used
- Microsoft Excel
- Microsoft PowerPoint
